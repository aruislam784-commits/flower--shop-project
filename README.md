🌸 Flower Shop Management System

 📖 Description
The Flower Shop Management System is a web-based application developed using HTML, CSS, JavaScript, PHP, and MySQL. It allows users to browse flowers, view product details, and contact the shop. The admin can manage products and customer information.

✨ Features
- User-friendly homepage
- Flower product listing
- Product details
- Contact form
- User login and registration
- Admin panel
- Add, Edit, and Delete products
- MySQL database integration
- Responsive design

 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

 🚀 How to Run
1. Install XAMPP.
2. Copy the project folder into the `htdocs` folder.
3. Start Apache and MySQL.
4. Import the database into phpMyAdmin.
5. Open your browser and visit:
   ```
   http://localhost/FlowerShop
   ```
